Summary

This project is been presented as a final task of Practical Machine Learning Course.

The goal is predict, using tools of machine learning, the variable class, that represents the way the participants of search do your movements, to keep health.

Two techniques were used to model the data: Classification trees and Random forests. After build a model with each technique and make a test to verify the accuracy of them, we made a prediction on data and got the results.
